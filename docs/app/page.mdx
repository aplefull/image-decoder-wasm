# Image Decoder

A browser-based image decoder library that uses WebAssembly to support image formats not natively supported by browsers.

## Supported Formats

| Format | Extension | Description |
|--------|-----------|-------------|
| AVIF | `.avif` | AV1 Image Format |
| HEIF/HEIC | `.heif`, `.heic` | High Efficiency Image Format |
| WebP | `.webp` | Google's image format |
| JPEG XL | `.jxl` | Next-generation JPEG image codec |
| JPEG | `.jpg`, `.jpeg` | Standard JPEG images |
| JPEG-LS | `.jls` | Lossless/near-lossless JPEG |
| TIFF | `.tiff`, `.tif` | Tagged Image File Format |
| RAW | `.raw`, `.cr2`, `.nef`, `.arw`, etc. | Raw image formats |
